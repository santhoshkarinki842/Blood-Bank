User Management Guidelines:
  • Activate: Allow users to login after registration
  • Deactivate: Prevent users from logging in
  • Make Admin: Grant administrative privileges
  • Remove Admin: Revoke administrative privileges
  • Delete User: Permanently remove a user account (cannot be undone)
